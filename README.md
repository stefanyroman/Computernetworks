# Computernetworks
group project 2026

The final project is group work, consisting of a software application developed in Python that leverages computer network technologies to solve a real-world problem. Therefore, students are required to form groups of at least three (3) students before submitting the proposal (see the “Group” section on Blackboard). Note that joining a group is required by the system in order to submit the proposal and, later, the final project. Individual work (i.e., group of one member) will be penalized by deducting 1.5 points of the grade. 



Submitting the proposal is always required in order to submit and present the final project in the next step. Following submission, the instructor will review the proposal and provide feedback regarding the final project the students intend to pursue. Late submissions will be accepted but not reviewed by the instructor, resulting automatically in a grade of zero (0). If the instructor will provide feedback, it will be the students' responsibility to incorporate it.



The project should be exclusively conceived for this course. Reusing existing software and/or source code from previous courses is not allowed. In the proposal, students should indicate the GitHub repository URL. This will enable the instructor to keep track of the changes as they are made, check the authenticity of the code, and evaluate the thinking process underlying it. The sequence of commits should clearly demonstrate the project’s coding progression.



Moreover, the project has a prominent interdisciplinarity requirement, which is usually the hardest part for students. This accounts for 8 out of 30 points awarded for the final project (for details, see the syllabus). Project interdisciplinarity requires students to implement an innovative algorithm inspired by principles that belong to non-computing disciplines (e.g., biology, psychology, etc.). Alternatively, students may consider solving a problem that belongs to non-computing disciplines using computer network technology.



If submitted within the deadline, a well-structured and clear proposal will result in five (5) points.
